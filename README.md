# 🌍 Disaster Preparedness App

A full-stack Disaster Preparedness Web Application designed to help users stay informed, prepared, and safe during emergencies.

This platform provides real-time alerts, safety guidelines, emergency contacts, and preparedness resources.

---

## 🚀 Features

* 📢 Real-time disaster alerts
* 📍 Location-based emergency updates
* 📋 Disaster safety guidelines
* 📞 Emergency contact directory
* 🗺️ Resource & shelter information
* 🔐 Secure backend API integration

---

## 🛠️ Tech Stack

### Frontend (Client)

* React.js / JavaScript
* HTML5
* CSS3

### Backend (Server)

* Node.js
* Express.js
* REST APIs

### Database

*  MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/disaster-preparedness-app.git
cd disaster-preparedness-app
```

### 2️⃣ Install Dependencies

For backend:

```bash
cd server
npm install
```

For frontend:

```bash
cd client
npm install
```

### 3️⃣ Run the Application

Start backend:

```bash
cd server
npm start
```

Start frontend:

```bash
cd client
npm start
```
